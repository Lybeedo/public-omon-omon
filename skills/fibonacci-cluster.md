# Fibonacci Cluster

Fibonacci Cluster adalah zona di mana **2+ level Fibonacci dari swing berbeda saling berdekatan**, menghasilkan area support/resistance lebih kuat dari single Fib level.

## Rasio Fibonacci Utama

| Rasio | Nilai | Penggunaan |
|-------|-------|------------|
| 0.236 | 23.6% | Retracement minor |
| 0.382 | 38.2% | Retracement umum |
| 0.500 | 50% | Mid-point (widely used) |
| 0.618 | 61.8% | Golden ratio — paling signifikan |
| 0.786 | 78.6% | Retracement dalam |
| 1.272 | 127.2% | Extension |
| 1.618 | 161.8% | Extension penting |

## Jenis Fibonacci untuk Dicluster

1. Retracement + Retracement — level dari dua swing berbeda
2. Retracement + Extension — pullback di zona extension
3. Retracement + Projection — ABCD pattern atau harmonic
4. Multi-timeframe — Fib dari Daily + H4 + H1 yang converge

## Cara Menemukan Fibonacci Cluster

### Langkah 1 — Identifikasi semua swing relevan
- Timeframe besar (Daily/H4) untuk swing utama
- Timeframe kecil (H1) untuk swing minor

### Langkah 2 — Tarik Fib retracement dari setiap swing
- Catat level: 23.6, 38.2, 50, 61.8, 78.6

### Langkah 3 — Cari area berdekatan
- Cluster valid jika level berjarak **≤ 15-20 pip** (FX) atau proporsional
- Minimal **2+ level berbeda** converge

### Langkah 4 — Konfirmasi dengan price action
- Candlestick reversal (pin bar, engulfing)
- Volume spike
- Rejection di zona

## Kombinasi Sinyal Kuat

- Fibonacci Cluster + RSI divergence
- Cluster + volume profile
- Cluster + supply/demand zone
- Cluster + candlestick reversal pattern
- Multi-timeframe confluence (2+ TF)

## Kelebihan vs Kelemahan

**Kelebihan:**
- Konfluensi tinggi → probabilitas tinggi
- Mengurangi sinyal palsu
- Cocok untuk grid trading & systematic EA

**Kelemahan:**
- More complex — banyak swing harus diidentifikasi
- Manual-intensive — sulit diotomasi sepenuhnya
- Subjective — swing relevan butuh pengalaman

## Aplikasi pada Harmonic Patterns

Fibonacci Cluster adalah dasar dari harmonic patterns:
- Gartley (AB=CD + cluster di X)
- Bat (extended BC + cluster)
- Crab (deep Crab + 88.6% Fib)
- Butterfly (extension 127%+ cluster)

## Contoh Penerapan

**Scenario: GOLD (XAUUSD) Uptrend**

```
Swing 1 (Daily): 1900 → 1950 → pullback ke 1938 (61.8%)
Swing 2 (H4):    1930 → 1980 → pullback ke 1936 (38.2%)
Swing 3 (H4):    1920 → 1970 → pullback ke 1939 (50%)

→ Zona 1936-1939 adalah Fibonacci Cluster
→ Golden pocket confluence zone
→ Area buy dengan SL di bawah 1930, TP di 1970+
```

## Catatan

- Subjective — swing high/low bisa berbeda antar trader
- Tidak berdiri sendiri — sebaiknya dikombinasikan dengan indikator lain
- False signal bisa terjadi saat sideways / low volatility
- **Bukan holy grail**, tapi alat bantu karena self-fulfilling prophecy
